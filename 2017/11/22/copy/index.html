Set formats using xlswriter

Powerful format tools: .write(), add_format, set_row(), set_column(), .merge_range()

  • .write(location,text,format)
    • location：目标单元格，such as 'A1'
    • text: 待输入信息
    • format: 单元格格式，参数包括加粗、对齐方式、字体、字号、颜色 workbook.add_format({'bold':True,'align': 'center','font_size':14,'font_name':'微软雅黑','font_color':'#7C140C'})
  • .write(row,column,text)
    • caution: row and column starts from 0
    • for text you can put enve Excel functions!
    • e.g. worksheet.write(4,0,'=SUM(A3:A4)')
  • .insert_image(locations, address of pics)
    • 插入图片
    • 图片地址可以是一个url，也可以是本地地址，类似于MARKDOWN e.g, worksheet.insert_image('B5','f:\1.jpg',{'url':'http://www.python.org'})
  • set_row(row,height,cell_format,options)
    • 此方法设置行单元格的属性,row指定行位置,height指定高度,单位是像素,
    • cell_format指定格式对象,参数options设置hiddeen(隐藏),level(组合分级),collapsed(折叠,例如: cell_format=workbook.add_format({'bold':True}) worksheet.set_row(0,40,cell_format) 设置第一行高40,加粗
  • set_column(first_col,last_col,width,cell_format,options)
    • 设置列单元格的属性,具体参数说明如上.worksheet.set_column(0,1,10) worksheet.set_column('C:D',20)
  • .merge_range(location,text,format)
    • 合并单元格
    • e.g.worksheet.merge_range('A2:B2','为QTA汪峰老师打call',format1)

FYI:

  • write(row,col,*args):写普通数据到工作表的单元格,row行坐标,col列坐标,起始都是以0开始,*args为写入的内容,可以是字符串,文字,公式等,writer方法已经作为其它更具体数据类型方法的别名
  • write_string():写入字符串类型,worksheet.write_string(0,0,'your text')
  • write_number():写入数字类型,worksheet.write_number('A2',1.1)
  • write_blank():写入空类型数据,worksheet.write_blank('A2',None)
  • wirte_formula():写入公式类型,worksheet.write_formula(2,0,'=SUM(B1:B5))
  • write_datetime():写入日期类型数据,worksheet.write_datetime(7,0,datetime.datetime.strptime('2014-01-02','%Y-%m-%d),workbook.add_format({'num_format':'yyyy-mm-dd'}));
  • write_boolean():写入逻辑类数据,worksheet.write_boolean(0,0,True)
  • write_url():写入超链接类型数据,worksheet.write_url('A1','ftp://www.python.org')
  • write_column():写入到一列,后面接一个数组
  • wirte_row():写入到一行,后面接一个数组

Adding Charts

  • chart = workbook.add_chart({'type':'column','subtype':''}) （和之前一样参数均以字典的形式传入）

  • types

    • area:创建一个面积样式的图表;
    • bar:创建一个条形样式的图表;
    • column:创建一个柱形样式的图表;
    • line:创建一个线条样式的图表；
    • pie:创建一个饼图样式的图表；
    • scatter:创建一个散点样式的图表；
    • stock:创建一个股票样式的图表;
    • radar:创建一个雷达央视的图表；

  • 向图表传输数据 chart.add_series({})

    • chart.add_series({
      'names':'数据范围',
      'categories':'数据范围',
      'values': '数据范围',
      'y2_axis': True,
      })

  • 通过worksheet.insert_chart(location,chart)添加
